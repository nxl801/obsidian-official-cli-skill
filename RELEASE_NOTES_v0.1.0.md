# Release Notes / 发布说明 — v0.1.0

## English

Initial public release of **Obsidian Official CLI Skill for OpenClaw**.

### Highlights
- Built for **Obsidian 1.12+ official CLI** (`obsidian`)
- Retrieval-first workflow for OpenClaw / Claude Code / AI agent usage
- Covers `search`, `search:context`, `read`, `backlinks`, `links`, `tags`, `properties`, `tasks`, and `unresolved`
- Includes packaged `.skill` artifact for direct use
- Validated and tested against a real Obsidian vault

### Positioning
This skill does **not** replace older community `obsidian-cli` / NotesMD workflows. It is a parallel skill for environments where the official Obsidian CLI is already enabled and preferred.

---

## 中文

这是 **Obsidian Official CLI Skill for OpenClaw** 的首个公开版本。

### 亮点
- 面向 **Obsidian 1.12+ 官方 CLI**（`obsidian`）
- 适合 OpenClaw / Claude Code / AI Agent 的 retrieval-first 工作流
- 覆盖 `search`、`search:context`、`read`、`backlinks`、`links`、`tags`、`properties`、`tasks`、`unresolved` 等核心命令
- 附带可直接使用的 `.skill` 打包产物
- 已完成校验，并在真实 Obsidian vault 上做过测试

### 定位说明
这个 skill **不是为了替换** 旧的社区 `obsidian-cli` / NotesMD 路线，而是为已经启用官方 Obsidian CLI 的环境提供一条更自然、更透明的 Agent 访问方式。
